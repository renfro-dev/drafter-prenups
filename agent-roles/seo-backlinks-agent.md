### Black Hat Tactics (Never Use)

**DO NOT:**
- âŒ Buy links from link farms or PBNs (Private Blog Networks)
- âŒ Participate in link exchanges / reciprocal linking schemes
- âŒ Use automated link building software
- âŒ Comment spam on blogs
- âŒ Submit to low-quality directories (500+ sites)
- âŒ Create fake guest post author profiles
- âŒ Use exact match anchor text excessively
- âŒ Get links from unrelated niches (gambling, adult, pharmaceuticals)
- âŒ Pay for links without "nofollow" attribute (violates Google guidelines)

**Why This Matters:**
- Google penalties can devastate rankings for months/years
- Manual actions are hard to recover from
- Brand reputation damage
- Wasted time and money

### Toxic Link Monitoring

**Monthly Audit:**
1. Run Ahrefs/Semrush backlink audit
2. Filter by spam score >30%
3. Identify toxic links:
   - Spammy directories
   - Foreign language sites (if not targeting those markets)
   - Adult/gambling/pharma sites
   - Hacked sites with malware
4. Disavow toxic links via Google Search Console

**Disavow File Format:**
```
# Disavow file for drafter.com
# Date: 2025-10-18
# Reason: Spammy backlinks

# Individual URLs
http://spammy-site.com/bad-link

# Entire domains
domain:another-spam-site.com
```
